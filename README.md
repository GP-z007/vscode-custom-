# vscode-custom-

So, install this extension Custom CSS and JS Loader

and go to settings.json file in vscode

create a css file and in settings.json file add this code :

```json
  "vscode_custom_css.imports": [
        "file:///Users/gokulprasath/Desktop/LPU/ob/vs-code.css"
    ],
```
